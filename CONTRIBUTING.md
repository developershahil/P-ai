# Contributing

Thanks for contributing to Personal AI.

## Development setup
1. Create a virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
   - `pip install -e .[dev]`
3. Run tests with `pytest -q`.

## Pull request checklist
- Keep source code under `src/personal_ai/`.
- Add/update tests under `tests/`.
- Update docs when behavior changes.
- Keep module names lowercase and class names CamelCase.
